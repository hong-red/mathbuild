## 主要内容

### 1. 数据获取与清洗
- 采集了北京、成都、武汉三座城市的年度发展数据
- 使用 Python 对原始数据进行清洗与合并

### 2. 数据分析与可视化
- 房地产专题分析
- 经济发展分析
- 人口与社会发展分析
- 房价收入比分析
- 影响因素占比分析

### 3. 数学模型
- **城市发展潜力评价**：基于客观权重（如熵权法）的综合评价模型
- **房价预测**：自回归模型（AR）进行房价时间序列预测

### 4. 论文撰写
- 使用 LaTeX 编写完整数学建模论文
- 包含摘要、模型建立、求解、验证等完整结构

## 技术栈

- **Python**：数据分析与建模
  - `pandas`：数据处理
  - `numpy`：数值计算
  - `matplotlib`：数据可视化
- **LaTeX**：论文排版

## 使用说明

1. 克隆仓库到本地
2. 安装依赖：`pip install pandas numpy matplotlib openpyxl`
3. 运行各目录下的 `.py` 文件即可复现分析与图表
