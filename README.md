## 各届赛题简介

### 25校赛 — LED 显示屏颜色设计与校正
- **赛题方向**：颜色科学、色域映射、RGB 校正
- **核心内容**：
  - 色域映射优化与可视化
  - 色度距离分析与色域压缩
  - RGB 及白色基底差异对比与复合校正
- **成果**：完整论文 + 可运行 Python 代码 + 运行截图

### 26校赛 — 城市发展与房价预测
- **赛题方向**：城市数据分析、综合评价、时间序列预测
- **核心内容**：
  - 数据获取与清洗（北京、成都、武汉三城市）
  - 数据分析与可视化（房地产、经济、人口等维度）
  - **城市发展潜力评价**：基于客观权重（如熵权法）的综合评价模型
  - **房价预测**：自回归模型（AR）进行房价时间序列预测
- **成果**：完整 LaTeX 论文 + Python 代码 + 数据与图表

## 技术栈

- **Python**：数据分析与建模
  - `pandas`：数据处理
  - `numpy`：数值计算
  - `matplotlib`：数据可视化
- **LaTeX**：论文排版（26校赛）

## 使用说明

1. 克隆仓库到本地
2. 安装依赖：`pip install pandas numpy matplotlib openpyxl`
3. 进入对应校赛目录，运行 `.py` 文件即可复现分析与图表
